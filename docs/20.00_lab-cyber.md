## general prework: Set Adminadccess via ssh keys from ansible host:
create key and copy
```bash
ssh-keygen -t rsa -b 4096 -N "" -f ~/.ssh/id_rsa
cat ~/.ssh/id_rsa
```
for every dd

press Ctrl-D
```bash

ssh sysadmin@192.168.1.32 adminaccess add ssh-keys user sysadmin 
ssh sysadmin@192.168.1.67 adminaccess add ssh-keys user sysadmin
```

## seting up repl contect from vault
change to a vault pairing, in this example vault ddve from replication ddve-02
```bash
cd ~/workspace/0499/lab_cyber/vault-ddve_ddve-02
```
list the mtrees:
```bash
ssh sysadmin@${SOURCE_DD_NAME} mtree list
```

for simple test, lets start with sysdr, use any other mtree you need
```bash
MTREE=/data/col1/SysDR-R_ppdm-1
```

setup context

```bash 
ssh ${SSH_EXEC} "replication add source mtree://${SOURCE_DD_NAME}${MTREE} destination mtree://${VAULT_DD_NAME}${MTREE}_repl"
ssh ${SSH_EXEC} "replication modify mtree://${VAULT_DD_NAME}${MTREE}_repl connection-host ${SOURCE_CONNECTION_HOST} port 2051"
ssh ${SSH_EXEC} "replication show config"
```

## setup and initialize context from the source
