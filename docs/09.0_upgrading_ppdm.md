# PPDM Update

copy the PPDM Update file to a Path on your System, here /swdist

Upload the File to PPDM:
```bash
ansible-playbook ~/workspace/ansible_ppdm/9.9-playbook_upload_update_ppdm_native.yml -e "upgrade_source=/swdist"
```
