## PPDM CLI Download with ORAS - Nutshell

### 🎯 **Download Method**

PPDM CLI uses **ORAS (OCI Registry As Storage)** to distribute binaries across platforms.

### 🔧 **Download Commands by Platform**

**Linux AMD64:**
```bash
oras pull quay.io/delldps/ppdm-cli:latest-linux-amd64
```

**Linux ARM64:**
```bash
oras pull quay.io/delldps/ppdm-cli:latest-linux-arm64
```

**Windows AMD64:**
```bash
oras pull quay.io/delldps/ppdm-cli:latest-windows-amd64
```

**macOS AMD64:**
```bash
oras pull quay.io/delldps/ppdm-cli:latest-darwin-amd64
```

**macOS ARM64 (Apple Silicon):**
```bash
oras pull quay.io/delldps/ppdm-cli:latest-darwin-arm64
```

**FreeBSD AMD64:**
```bash
oras pull quay.io/delldps/ppdm-cli:latest-freebsd-amd64
```

### 📋 **What ORAS Does**

**1. Registry Connection:**
- Connects to OCI registry (quay.io)
- Authenticates if needed
- Downloads platform-specific binary

**2. Binary Extraction:**
- Extracts PPDM CLI binary from OCI layer
- Saves as `ppdm-cli` (or `ppdm-cli.exe` on Windows)
- Makes executable (on Unix systems)

**3. Version Management:**
- Supports version tags (e.g., `20.1.0.0`)
- Supports `latest` tag
- Platform-specific tags ensure correct binary

### 🎯 **Key Benefits**

- **Single Tool** - ORAS handles all platforms
- **Standard Registry** - Uses standard OCI registry
- **Version Control** - Tag-based version management
- **Platform Specific** - Correct binary for each architecture

### 💡 **Simple Workflow**
```bash
# 1. Install ORAS
curl -LO https://github.com/oras-project/oras/releases/download/v0.16.0/oras_0.16.0_linux_amd64.tar.gz
tar -xzf oras_0.16.0_linux_amd64.tar.gz

# 2. Download PPDM CLI
oras pull quay.io/delldps/ppdm-cli:latest-linux-amd64

# 3. Use it
./ppdm-cli version
```

**Bottom line: ORAS pulls platform-specific PPDM CLI binaries from OCI registry with simple, standardized commands.**
