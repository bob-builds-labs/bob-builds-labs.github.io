# How to upgrade openshift from CLI

## View available Versions

```bash
oc adm upgrade
```

## Ubprade to next Minor

```bash
oc adm upgrade | grep quay.io | tail -n1 | awk '{print $1}' | xargs -I{} oc adm upgrade --to={}
```
